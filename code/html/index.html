Présentation du site

Notre site internet a été conçue pour expliquer l'évolution du développement web à travers le temps de la premiere version 1.0 à la dernière version 4.0 prévue pour 2020, ainsi qu'une projection dans le futur du web (5.0).Notre projet à été proposé par Monsieur Amanton et va être réalisé par Thomas Vanrokeghem, Hugo Labbé et Yann Cornu
