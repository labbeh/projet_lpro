Le Web 0.0: les origines du Web

Le web 0.0, est un concept qui a été définit par certains comme la base du fonctionnement du web, c'est à dire la mise en place de l'architecture ARPANET et du protocole TCP/IP. Néanmoins, ce concept de web 0.0 n'est pas admis par tous: certains pensent qu'il faut commencé à considérer le web à partir du web 1.0 dans la mesure ou ce qui est définit comme étant le web 0.0 n'est que d'un bas niveau technique et ne met pas en avant un concept de comportements sur son utilisation.

Le Web 1.1: première version grand public

La première version grand public du web était purement statique, c'est à dire que l'internaute ne pouvait que consulter le contenu mis en ligne par l'administrateur du site. Au début, mes moteurs de recherches étaient inexistant il fallait donc connaitre l'URL du site pour pouvoir y accéder. Les pages des sites étaient souvent codées "en dure" avec les première version d'XML, HTML et XHTML. Les premières version de PHP et Java WEB ont permis de rendre les pages dynamiques et les premiers CMS sont apparus. Un CMS permet de créer des pages web sans toucher directement au code html ce qui permet de générer un code HTML "sur mesure" en fonction de la requête effectuée. Néanmoins, le web n'était toujours pas interactif avec l'internaute, la relation producteur consommateur était toujours de mise.
