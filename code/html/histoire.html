Le web: les origines

Le web ou "World Wide Web" (= la toile d'araignée mondiale en français), est un système sur internet permettant de consulter des pages dans un navigateur. Le web repose sur le protocole HTTP et sur un système de liens hypertexts pour faire le liens entre les différentes pages, d'ou l'expression "toile d'araignée" puisque l'ensemble des liens sur le web forment une toile. Le web est la partie d'internet la plus utilisée du grand public. En effet, à partir du moment ou nous ouvrons un navigateur pour consulter un site, nous sommes sur le web

Un versionning non prévu à l'avance

Les versions du web que nous détaillons sur ce site, n'ont rien à voir avec par exemple, la version d'un logiciel. En effet, lorsque l'on sort un logiciel dans une certaines version (exemple v2.0.1), ce numéro de version est prévu à l'avance et décidé par les développeurs

Il n'en ai rien de cela pour les versions du web: ici, les versions ne sont pas décidées à l'avance mais déduite par des spécialistes du web, par observation de ce qui s'y passe. Même si certaines futur versions commencent à être prédites avant leur mise en place, elles ont été déduite de part l'évoltion courante des comportements sur le web.
